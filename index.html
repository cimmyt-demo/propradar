How to use this demo

This static prototype behaves like a full PropFirmMatch-style product running entirely in your browser. Nothing is saved to a server — everything uses localStorage and demo data.

  1. 1. Create an account (optional but recommended). Visit the Auth page, sign up, and you’ll unlock favorites, a personal dashboard and a real loyalty balance.
  2. 2. Browse firms & lists. Go to Firms or Prop Firm Lists, filter by assets/models/platforms and open firm detail pages.
  3. 3. Compare challenges. From the directory or firm pages, add firms to comparison and open the Compare view for a rules-focused table.
  4. 4. Simulate buys & earn points. Click any “Buy via partner” button to log a simulated affiliate click, then head to Loyalty and submit a proof to earn demo points.
  5. 5. Leave reviews. Use Reviews to submit your own local-only feedback on firms and see how a verified-style system would look.
  6. 6. Inspect analytics. Finally, open the Admin view to see clickouts, analytics events and reviews generated from your demo session.
